My Journey into Web Development

Published on January 1, 2025

How It All Began

During my time at Berry University, I delved into courses such as Object-Oriented Programming, Web Engineering, and Artificial Intelligence. These courses laid the foundation for my understanding of programming and sparked my interest in web development.

The First Steps

My journey began with learning HTML and CSS, where I built simple webpages to practice structuring content and styling it. One of my earliest projects was creating a portfolio to showcase my work, which evolved as my skills grew.

Key Challenges

One of the biggest challenges was understanding how all the pieces of web development fit together. Balancing structure, style, and functionality was daunting at first, but with perseverance, I began to see how tools like JavaScript enhanced user interaction.
